# Security Policy

## Supported Versions

We actively support and patch the latest version of Pandereta.

| Version | Supported          |
| ------- | ------------------ |
| 5.0.x   | :white_check_mark: |
| < 5.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a vulnerability or security risk, please do **not** open a public issue. Instead, report it responsibly:

1. Send an email to **security@example.com** with details of the vulnerability.
2. Include a detailed description, steps to reproduce, and any proof of concept or screenshots.
3. We will acknowledge your report within 48 hours and work on a resolution.

Thank you for helping keep this project secure!
